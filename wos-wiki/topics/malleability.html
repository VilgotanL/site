Malleability

Malleability is the maximum size of a part. Parts can also be fixed-size, such as extractors.

Malleability data

Running game:GetObjects("rbxassetid://6724254977")[1].Parent = workspace in the command bar in Roblox Studio gives you the source code of the MBTools plugin, which has a script with malleability data.
Some parts (that aren't in MBTools) are not present, like DevBattery, DevGenerator, tinnitus, etc
